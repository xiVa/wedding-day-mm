# Wedding Day · Marion & Mickaël

Site du mariage de Marion & Mickaël — 23 août 2026 · Quinta De Vale Minhoto, Portugal.

## Contenu

- **`index.html`** — site public pour les invités (programme, infos, accès, contact, photos WedShoots)
- **`planning.html`** — fiche conducteur interne pour Joba (planning minuté, checklist synchronisée temps réel)

## Stack

- Site statique HTML/CSS/JS (zéro build)
- Firebase Firestore pour le contenu dynamique (programme) et la checklist temps réel
- Hébergement Netlify avec déploiement continu depuis GitHub

## Déploiement

Tout commit sur la branche `main` est déployé automatiquement sur [wedding-day-mm.netlify.app](https://wedding-day-mm.netlify.app).
